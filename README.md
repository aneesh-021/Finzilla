# Finzilla

**Tagline:** *From Transactions to Intelligent Business Decisions.*

## Overview

Finzilla is an AI-powered financial intelligence platform designed for small businesses. It consolidates financial data from multiple sources such as bank statements, UPI transactions, accounting software, Excel files, and invoices into a single platform. By transforming scattered financial data into meaningful insights, Finzilla enables business owners to monitor financial health, analyze cash flow, track profitability, identify spending patterns, and make informed business decisions.

The platform follows a simple philosophy:

**Collect → Organize → Analyze → Explain → Predict**

Reliable data collection and organization form the foundation for analytics, business intelligence, and AI-powered recommendations.

## Problem

Small business owners often maintain financial records across different systems. Since this information is fragmented, understanding the overall financial position requires manual effort, increasing the chances of errors and delaying decision-making.

## Solution

Finzilla provides a centralized platform to upload, validate, clean, store, analyze, and visualize financial transactions. It generates financial reports, dashboards, and AI-assisted business summaries that help owners understand their business performance.

## MVP Features

* Upload Bank Statement (CSV)
* Upload UPI Transactions (CSV)
* Data Validation and Cleaning
* Transaction Storage
* Revenue Analysis
* Expense Analysis
* Profit Analysis
* Cash Flow Analysis
* Financial Dashboard

## Technology Stack

* Backend: Python, FastAPI
* Data Processing: Pandas, NumPy
* Analytics Database: DuckDB
* Future Production Database: PostgreSQL
* Data Warehouse: Snowflake
* Machine Learning: Scikit-learn, Prophet, XGBoost
* OCR: EasyOCR
* AI: LLM API
* Dashboard: Power BI
* Version Control: Git & GitHub

## Project Structure

* `docs/` – Documentation

* `data_pipeline/` – ETL pipeline
* `analytics/` – Financial analytics modules

* `ocr/` – Invoice and receipt processing
